# Suite-de-Analisis-Biomedico-y-Reportes-Estadisticos-BioMed-Analyzer-
Aplicacion de escritorio en Python para diagnostico clinico y evaluacion metabolica. Posee una arquitectura desacoplada en capas (Tkinter/Matplotlib) que procesa y grafica en tiempo real perfiles lipidicos, glucemia, anemia, metabolismo e indices antropometricos segun estandares internacionales de la OMS y la ADA.
